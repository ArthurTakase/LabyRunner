LabyGame Beta 1.2

------ Avant de pouvoir jouer ------

Avoir installé Python sur son ordinateur (https://www.python.org/downloads/)
Avoir les librairies suivantes :
>>> Pygame (pip install pygame)
>>> pyautogui (pip install pyautogui)

----- Lancer le jeu -----

Double cliquer sur le fichier '__main__.py'

----- Comment jouer -----

Toutes les infos se trouvent dans le menu 'Tuto' du jeu.
